# Video Downloader API

API para download de vídeos do YouTube, SoundCloud, Instagram, TikTok e Twitter.

## Instalação

```bash
npm install
```

## Executando

```bash
node index.js
```

# Endpoints:

## Download:
`/api/download?url=https://...`

## Puxar informações:
`/api/get-info?url=https://...`
